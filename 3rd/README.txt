UoB SSH VPN by KevinZonda

Content under this folder are 3rd binaries.

- plink by Simon Tatham licensed under MIT (file uoblink.exe)
- Privoxy by Privoxy Developers licensed under GNU GPLv2 (file privoxy.exe)

Move this folder to the same directory of laucher, then
rename this folder to bin.

E.g.

- KevinZonda.UoB.SshVPN.exe
- KevinZonda.UoB.SshVPN.dll
  + bin
    + conf
    + privoxy.exe
    + uoblink.exe
